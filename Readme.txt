# About CrazyTremor: 
# 	A MATLAB GUI (graphical user interface) software for examining for triggered tectonic tremor
# 	This package is developed by Kevin Chao (email: kchao@northwestern.edu) and Chunquan Yu (email: yucq@gps.caltech.edu) 

# Reference paper:
# Chao, K. and C. Yu, A MATLAB GUI for Examining Triggered Tremor: A Case Study in New Zealand (2018), Seismol. Res. Lets., in revision.

# A quick start:
# 	3.1 Download demonstration SAC files here: https://www.dropbox.com/s/yv4trvy1vsreui9/demo_20120411_Japan.zip?raw=1
# 	3.2 Put the SAC folder under ~/CrazyTremor_2018_v1.1/SAC/
# 	3.3 In Matlab (2017), run: ct_gen_listname
# 	3.4 In Matlab, run: y_CrazyTremor_v1
# 	3.5 In the pop-up window, click “Reset map (m)”
